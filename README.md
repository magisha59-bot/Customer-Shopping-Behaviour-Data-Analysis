Customer Shopping Behavior Analysis
Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The goal of the analysis is to identify patterns in customer spending, product preferences, subscription behavior, and purchase frequency to support data-driven business decisions.

The project combines Python for data cleaning and exploration, SQL for business analysis, and Power BI for visualization, demonstrating an end-to-end data analytics workflow.

Project Workflow
1. Data Cleaning and Exploration (Python)

Python was used for preprocessing and preparing the dataset for analysis.

Key Steps

Data Loading

Imported dataset using pandas.

Initial Exploration

Used df.info() to inspect data structure

Used df.describe() for statistical summaries.

Handling Missing Values

Missing values in review_rating were filled using the median rating within each product category.

Column Standardization

Renamed columns to snake_case for better readability and consistency.

Feature Engineering

Created age_group by binning customer ages.

Created purchase_frequency_days to standardize purchase frequency metrics.

Data Consistency Checks

Identified redundancy between discount_applied and promo_code_used.

Removed the promo_code_used column.

Database Integration

Cleaned dataset was loaded into PostgreSQL for advanced SQL analysis.

Key Insights

Certain age groups contribute significantly more revenue, indicating strong target segments.

Subscribers tend to spend more compared to non-subscribers.

Some products rely heavily on discount-driven sales.

Express shipping users show higher purchase values compared to standard shipping.

Loyal customers contribute a large portion of revenue, highlighting the importance of retention strategies.

Business Recommendations

Based on the analysis, the following actions are recommended:

1. Increase Subscription Adoption

Promote subscription programs with exclusive discounts and benefits.

2. Strengthen Customer Loyalty Programs

Reward repeat customers to convert returning buyers into loyal customers.

3. Optimize Discount Strategy

Ensure discounts are applied strategically to maximize revenue without reducing profit margins.

4. Promote High-Rated Products

Highlight top-rated and best-selling products in marketing campaigns.

5. Target High-Value Customer Segments

Focus marketing efforts on high-revenue age groups and express shipping users.
